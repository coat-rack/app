:::note
Reference and experimentation purposes there are a few example apps hosted within the [@coat-rack/app repository](https://github.com/coat-rack/app). These apps are available for installation via [UNPKG](https://unpkg.com/)
:::

The available apps can be downloaded using the following URLs

- `@coat-rack-catalog/sample-app` - `https://unpkg.com/@coat-rack-catalog/sample-app@<VERSION>/dist/`
- `@coat-rack-catalog/tasks` - `https://unpkg.com/@coat-rack-catalog/tasks@<VERSION>/dist/`
- `@coat-rack-catalog/notes` - `https://unpkg.com/@coat-rack-catalog/notes@<VERSION>/dist/`
